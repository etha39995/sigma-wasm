About Function Calling Agent

This demo showcases a client-side autonomous agent with local LLM inference and function calling capabilities. The agent can use tools to accomplish tasks.

Architecture: Uses a small LLM (DistilGPT-2) running in the browser via Transformers.js, with WebAssembly tools for function execution. The agent can call functions like calculate, process_text, and get_stats to solve problems.

Function Calling: The agent analyzes the goal, decides which tools to use, calls them via WASM, and uses the results to generate a final response. All processing happens client-side.

Available Tools:

  • calculate(expression): Evaluate mathematical expressions
  • process_text(text, operation): Process text (uppercase, lowercase, reverse, length, word_count)
  • get_stats(data): Get statistics from data arrays
