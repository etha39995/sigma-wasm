About Fractal Chat

This demo showcases an interactive chat interface that generates fractal images based on keywords in your messages.

How it works: Type a message containing a fractal keyword, and a corresponding fractal image will be generated and displayed in the chat. If no keyword is detected, the chat model will respond with text.

Available Fractal Keywords:

  • fractal - Random fractal from all types
  • mandelbrot - Classic Mandelbrot Set
  • julia - Julia Sets
  • buddhabrot or nebulabrot - Buddhabrot/Nebulabrot
  • orbit-trap - Orbit-Trap Fractals
  • gray-scott, reaction, or diffusion - Gray-Scott Reaction-Diffusion
  • l-system, tree, or plant - L-System Fractals
  • flames - Fractal Flames
  • strange, attractors, lorenz, clifford, or de jong - Strange Attractors
